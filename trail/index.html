What is Trail?

Trail is a browser-based path puzzle in the same family as the LinkedIn Zip game. You draw one unbroken line that visits the numbered checkpoints in order — 1, then 2, then 3 — while passing through every single cell exactly once. Pure logic, a single solution, no guessing.

Unlike puzzles capped at one per day, Trail is unlimited — tap "New puzzle" any time, or take the shared Daily Challenge and build a streak.

How to play

  1. Press on 1 and drag your finger or mouse to trace a line.
  2. Pass through the numbers in order (1 → 2 → 3 → …).
  3. Your line must cover every cell and never cross itself.
  4. Drag backwards to undo, or tap a cell on the line to rewind.

Tips to solve faster

Trail looks freeform, but it's pure deduction — there's exactly one path. A few habits make it click:

  • Start at the corners. A corner has only two neighbours, so the line must pass straight through it — that often fixes connections instantly.
  • Never strand a cell. Every cell must be visited; avoid moves that leave a cell with no way in or out.
  • Plan each number segment. The stretch between two checkpoints has to cover a set region without blocking the next leg.
  • Work edges inward. Borders have the fewest options, so committing them early narrows the middle.
